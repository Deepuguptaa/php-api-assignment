# PHP API Assignment

This project is a simple Product Inventory REST API built using Core PHP.

## Features

- Create Product API
- Get Product API
- Update Product API
- JSON file storage
- Input validation
- Proper HTTP status codes

## APIs

### 1. Create Product
POST /products

### 2. Get Product
GET /products/{id}

### 3. Update Product
PUT /products/{id}

## Technologies Used

- PHP
- JSON
- REST API

```bash
php -S localhost:8000
